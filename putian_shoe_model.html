三、核心计算公式

考虑退货后，每 1 个有效订单需要卖出 1/(1-退货率) 双（部分会被退回，鞋子可再售）

t = 1 / (1 - 退货率) // 退货15%时 t=1.176 运营商利润/有效单 = 售价 - 鞋价 - (售价÷ROAS)×t + 补贴×t - 售价×佣金率×t - 退货运费×(t-1) - 出库运费×t 总运营利润/有效单 = (鞋价-采购成本)×t + (售价÷ROAS)×t×返点率 - 补贴×t // 广告返点 = 广告费×35%，广告补贴 = 10元×t（每笔成交都给）

关键结构性问题

你的广告利润 = (P÷3) × 35% = P × 11.67%。当 P=70 时，广告返点仅 8.17元，但你要给 10元补贴。
广告返点 < 广告补贴 → 广告侧每单亏 1.83元，必须靠鞋价差额覆盖。
